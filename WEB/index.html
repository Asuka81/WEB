How To Use

How to adjust time ?

Press the adjust time button and press again to select what you want to adjust and then use up and down button to change the time.


How to set the alarm. 

Press the set alarm button then press again to select what you want to adjust alarm has 2 states which is active and deactive if you want to make the alarm alert set it to active.
